# Differential-gene-expression-analysis-across-breast-cancer-subtypes
This project performs differential gene expression analysis across  four breast cancer subtypes (Luminal A, Luminal B, HER2 and  TNBC) using microarray data from GEO dataset GSE65194. Six  key genes (TP53, CDK4, CCND1, BAX, BCL-2, CASP3) were analyzed using the LIMMA pipeline followed by visual plots in R
